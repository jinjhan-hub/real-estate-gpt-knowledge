隱私權政策

服務名稱：房仲 GPT Knowledge 助理

營運單位：太平洋房屋員林站前加盟店／九意開發有限公司

最後更新日期：2026 年 5 月 9 日

本隱私權政策說明「房仲 GPT Knowledge 助理」在使用 GPT Actions、API 或相關工具時，可能如何處理使用者提供的資料。 本服務主要用途為協助房仲產製銷售圖卡文案、社群靜態文案、周邊機能整理、手繪格局圖轉換、房市觀點文、稅務試算器規格與 API / OpenAPI 檢查。

一、我們可能收集或處理的資料

當使用者主動提供資料時，本服務可能處理以下內容：

  • 物件基本資料，例如案名、地址、坪數、格局、屋齡、開價、特色等。
  • 使用者上傳的物件照片、格局圖、名片或截圖。
  • 周邊機能查詢所需的地址、經緯度、查詢半徑與查詢類別。
  • 使用者輸入的文案需求、風格選擇、平台需求或其他任務說明。
  • API 或 OpenAPI 測試時所需的請求資料。

二、資料使用目的

上述資料僅用於以下目的：

  • 協助整理房仲物件資訊。
  • 協助產製銷售圖卡文案、FB / IG / Threads 靜態社群文案。
  • 協助查詢或整理指定地址周邊生活機能。
  • 協助手繪格局圖辨識與轉換。
  • 協助房市觀點文、稅務試算器或 API / OpenAPI 相關內容整理。
  • 協助 GPT Actions 與外部 API 正常運作。

三、資料分享與第三方服務

本服務可能透過 GPT Actions 呼叫外部 API，例如周邊機能查詢 API。 當使用者使用這些功能時，必要的查詢資料可能會傳送至對應 API 服務，以完成使用者要求的查詢或整理任務。

除完成使用者要求之功能外，本服務不會主動出售、出租或交換使用者提供的個人資料。

四、資料保存

本服務本身不主動建立會員系統，也不主動建立使用者個人資料庫。 若外部 API、GitHub Pages、OpenAI 或其他第三方平台因系統運作、資安或紀錄需求而保存技術紀錄，將依各該平台之政策辦理。

五、使用者注意事項

使用本服務時，請避免輸入不必要的敏感個人資料，例如：

  • 身分證字號
  • 完整戶籍資料
  • 銀行帳戶
  • 信用卡資料
  • 不必要的私人聯絡資訊
  • 未經當事人同意的個人資料

六、房仲資料與物件資訊聲明

本服務產生之內容僅供房仲行銷、文案整理、資訊輔助與內部作業參考。 實際物件資訊、價格、坪數、建物登記、稅務、法規、成交行情與周邊設施，仍應以正式謄本、合約、政府資料、稅務機關、主管機關或使用者自行查證結果為準。

七、稅務與法規聲明

若本服務協助整理房地合一稅、土地增值稅或其他不動產相關稅務內容，其結果僅供試算與參考，不構成正式稅務、法律或投資建議。 實際稅額與法律責任仍應以國稅局、地方稅務局、地政機關、專業律師、會計師或合格專業人士之認定為準。

八、隱私權政策更新

本政策可能因服務功能、API 串接方式、平台規則或法規要求而更新。 更新後將公布於本頁面，並以頁面所載最後更新日期為準。

九、聯絡方式

若對本隱私權政策或資料處理方式有疑問，請透過本服務營運者公開提供之聯絡方式聯繫。
