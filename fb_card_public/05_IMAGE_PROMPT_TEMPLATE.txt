# FB 圖卡圖片生成提示詞模板｜公版

使用時請根據已確認資訊替換括號內容。

生成一張 Facebook 4:5 房仲銷售圖卡。

風格：{風格名稱，例如高級豪宅風 / 溫暖家庭風 / 乾淨白底商務風}

限制：
1. 只能使用使用者提供的物件照片、名片與人物形象照。
2. 不要自行生成建物外觀、室內照片、街景或地圖。
3. 不要自行補價格、坪數、地址、電話、學區、交通距離。
4. 人物若來自名片或形象照，需保留五官、臉型、身形與辨識特徵，不得變臉。
5. 文字要大，手機上清楚可讀。
6. 主標 16 字以內，賣點最多 3 個。
7. 整體像房仲銷售圖卡，乾淨、清楚，不要過度浮誇。
8. 必須放入使用者已確認的經紀業名稱、經紀人姓名、經紀人證號。
9. 若上述三項揭露資訊未確認，不得生成圖片。

────────────────────
QR Code 版面硬限制
────────────────────

QR Code 一律不生成。

不得出現：
- QR Code 圖形
- LINE QR Code
- 條碼
- 類似掃碼圖樣
- 大型 QR Code 卡片
- 大型 QR Code 色塊

只能在「人物聯絡資訊區右下角」保留小型 QR Code 放置位。

尺寸限制：
- 寬度不得超過整張圖寬度 10%
- 高度不得超過人物聯絡資訊區高度 30%
- 不得大於電話號碼區
- 不得大於人物照片寬度的 35%
- 不得形成獨立大型區塊
- 不得成為視覺焦點

呈現方式：
- 細框
- 淡灰或淡米色框線
- 小字
- 留白
- 不使用厚重底色
- 不使用大面積黑底或紅底

區塊文字只能顯示：
QR Code預留區

或：
加入官方LINE

QR Code 區域用途：
僅供後續人工貼上，不是圖卡主視覺元素。

圖卡內容：
主標：{主標}
副標：{副標}
價格：{價格，若未提供則不要放}
賣點：
1. {賣點一}
2. {賣點二}
3. {賣點三}

下方聯絡資訊必須包含：

營業員姓名：{營業員姓名，若有辨識到必須放}
電話：{電話，若有辨識到必須放}
店名：{店名，若有辨識到必須放}
經紀業名稱：{經紀業名稱，必須放}
經紀人姓名：{經紀人姓名，必須放}
經紀人證號：{經紀人證號，必須放}
營業員證號：{營業員證號，若有辨識到必須放}

若營業員證號已辨識到，不得省略。

必放揭露資訊：
經紀業名稱：{使用者提供並確認的經紀業名稱}
經紀人姓名：{使用者提供並確認的經紀人姓名}
經紀人證號：{使用者提供並確認的經紀人證號}

版面：
上方放主標與副標。
中間使用物件照片。
下方放人物形象照、姓名、電話、店名、經紀業名稱、經紀人姓名與證號。
QR Code 預留區只能作為人物聯絡資訊區右下角的小型輔助區塊，不得獨立成大型卡片。
整體排版清楚、留白適中、文字層級明確。
聯絡資訊區不得只放電話與姓名。
若已辨識到營業員證號，必須出現在圖卡底部資訊區。
若已辨識到經紀人證號，也必須出現在圖卡底部資訊區。
兩者不得混淆、不得互相取代。
