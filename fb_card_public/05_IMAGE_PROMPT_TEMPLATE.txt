# 圖片生成提示詞模板｜照片保真版

任務：
生成一張 FB 4:5 台灣房仲銷售圖卡。

────────────────────
【最高限制｜照片來源】
────────────────────

圖卡內所有照片型畫面，只能來自使用者本次對話實際上傳的圖片。

嚴禁生成任何使用者未提供的照片型畫面。

禁止出現：
- AI 生成的房屋外觀
- AI 生成的室內照
- AI 生成的客廳、廚房、臥室、浴室
- AI 生成的街景
- AI 生成的社區大樓
- AI 生成的庭院、車庫、陽台
- AI 生成的地圖
- AI 生成的生活圈示意照片
- AI 生成的背景物件照
- AI 自行補出的第二張、第三張照片

若使用者只提供 1 張物件照片：
只能使用這 1 張照片。
不得把版面補成多張照片拼貼。

若使用者提供 2 張照片：
最多只能使用這 2 張。
不得新增第 3 張照片。

若照片不足：
使用純色背景、漸層色塊、資訊卡片、線條、icon、留白與文字排版補足。
不得用照片型畫面補足。

若使用者沒有提供任何物件照片：
不得生成正式銷售圖卡。

────────────────────
【可使用素材】
────────────────────

只能使用：
1. 使用者提供的物件照片
2. 使用者提供的名片
3. 使用者提供的人物照
4. 使用者確認的文字資料
5. verifyStoreAccess disclosure
6. 已確認的營業員與聯絡資訊

禁止自行補：
價格、坪數、地址、學區、交通距離、生活機能、電話、證號、照片、人物、QR Code。

────────────────────
【圖卡內容】
────────────────────

比例：
FB 4:5

主標：
{主標，16字內}

副標：
{副標}

價格：
{有提供才放，未提供則不要放}

賣點：
1. {賣點一}
2. {賣點二}
3. {賣點三}

風格：
{使用者選擇的風格}

────────────────────
【底部揭露資訊】
────────────────────

必須清楚放入：

經紀業名稱：{disclosure.brokerageName}
經紀人姓名：{disclosure.brokerName}
經紀人證號：{disclosure.brokerLicenseNo}

若有提供：
營業員：{營業員姓名}
營業員證號：{營業員證號}
電話：{電話}
LINE：{LINE ID}

不得省略已確認的營業員姓名與營業員證號。
不得用營業員資料取代經紀人資料。

────────────────────
【人物保真限制】
────────────────────

人物只能使用使用者本次對話實際上傳的人物照、名片人物照或原始照片裁切。

人物不得由 AI 重新生成。

禁止：
- 重新生成新人物
- 重畫人物臉
- 換臉
- 變臉
- 美化成不同長相
- 改五官
- 改臉型
- 改眼睛大小
- 改鼻子
- 改嘴型
- 改髮型
- 改膚色
- 改身形
- 卡通化
- 插畫化
- 3D 化
- 生成新的房仲人物

人物只能以「原圖裁切貼入」方式呈現。

若無法保持人物與原圖一致：
不得放人物。
改用純文字聯絡資訊區、人物照片預留區，或請使用者後續自行貼上人物照。

不得生成相似人物。
不得把名片人物重畫成新的真人。
不得為了版面完整而補一位房仲人物。

────────────────────
〖QR Code 版面硬限制〗
────────────────────

每張正式 FB 銷售圖卡都必須保留 QR Code 放置區。

QR Code 一律不生成。
這是「預留空位」，不是生成 QR Code。

不得出現：
- QR Code 圖形
- LINE QR Code
- 條碼
- 類似掃碼圖樣
- 黑白方格
- 可被誤認為 QR Code 的圖形
- 大型 QR Code 卡片
- 大型 QR Code 色塊

QR Code 預留區只能放在：
- 下方聯絡資訊區右側
- 或下方聯絡資訊區右下角
- 優先放在人物聯絡資訊區右下角

尺寸限制：
- 寬度不得超過整張圖寬度 10%
- 高度不得超過人物聯絡資訊區高度 30%
- 不得大於電話號碼區
- 不得大於人物照片寬度的 35%
- 不得大於營業員資訊區
- 不得形成獨立大型區塊
- 不得成為視覺焦點
- 不得遮住經紀業名稱、經紀人姓名、經紀人證號、營業員姓名、營業員證號、電話或 LINE

呈現方式：
- 小型正方形或接近正方形方框
- 細框
- 淡灰色、淡米色或淺色框線
- 白底或淺色底
- 小字
- 留白
- 不使用厚重底色
- 不使用大面積黑底、紅底、金色底或高彩度色塊

區塊文字：
方框內可留白，或只顯示以下其中一種文字：

QR Code預留區

或：

加入官方LINE QR Code

區域用途：
僅供後續人工貼上 QR Code。
不是圖卡主視覺元素。
不得為了 QR Code 預留區犧牲主要物件照片、主標、價格、賣點或法定揭露資訊。

若版面空間不足：
優先縮小裝飾元素、icon、副標或留白。
不得省略 QR Code 預留區。
不得放大 QR Code 預留區。
不得生成 QR Code 圖樣。

────────────────────
【版面建議】
────────────────────

上方：
主標、副標、價格

中間：
使用者提供的真實物件照片。
若照片不足，用色塊、資訊卡、icon 與文字排版補足。
不得新增照片。

下方：
最多 3 個賣點、聯絡資訊、人物照或文字聯絡區、小型 QR Code 預留區。

底部：
經紀業名稱、經紀人姓名、經紀人證號、營業員資訊。

整體要求：
文字大、清楚、手機可讀。
版面乾淨。
不得出現任何未提供照片。
