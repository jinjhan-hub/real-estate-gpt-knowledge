# 圖片生成提示詞模板

任務：
生成一張 FB 4:5 台灣房仲銷售圖卡。

必須使用：
1. 使用者提供的物件照片
2. 使用者確認的文字資料
3. 使用者提供的名片或人物照
4. verifyStoreAccess disclosure
5. 已確認的營業員與聯絡資訊

禁止：
自行生成房屋、室內、街景、地圖、人物、QR Code。
自行補價格、坪數、地址、學區、交通距離、電話、證號。

圖卡內容：
主標：{主標，16字內}
副標：{副標}
價格：{有提供才放}
賣點：
1. {賣點一}
2. {賣點二}
3. {賣點三}

底部資訊必放：
經紀業名稱：{disclosure.brokerageName}
經紀人姓名：{disclosure.brokerName}
經紀人證號：{disclosure.brokerLicenseNo}

若有：
營業員：{營業員姓名}
營業員證號：{營業員證號}
電話：{電話}
LINE：{LINE ID}

人物：
只能使用原圖裁切，不得重畫或變臉。

QR Code：
只放小型「QR Code 預留區」，不得生成掃碼圖樣。

風格：
{使用者選擇的風格}

版面：
上方主標，中間真實物件照片，下方賣點與聯絡資訊，底部揭露資訊。
文字需大、清楚、手機可讀。
