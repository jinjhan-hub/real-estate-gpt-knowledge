# 4:5 銷售圖卡圖片生成提示詞模板｜公版

使用時請根據已確認資訊替換括號內容。

生成一張 4:5 房仲銷售圖卡。

風格：{風格名稱，例如高級豪宅風 / 溫暖家庭風 / 乾淨白底商務風 / 買方決策輔助風 / 精品簡報風}

────────────────────
一、生成硬限制
────────────────────

1. 只能使用使用者提供的物件照片、名片與人物形象照。
2. 不得自行生成建物外觀、室內照片、街景、地圖或任何未提供畫面。
3. 不得自行補價格、坪數、地址、電話、學區、交通距離、周邊資訊。
4. 主標 16 字以內。
5. 賣點最多 3 個。
6. 文字要大，手機上清楚可讀。
7. 整體像房仲銷售圖卡，乾淨、清楚、不過度浮誇。
8. 必須放入使用者已確認的：
   - 經紀業名稱
   - 經紀人姓名
   - 經紀人證號
9. 若上述三項揭露資訊尚未確認，不得生成圖片。

────────────────────
二、人物形象照硬限制
────────────────────

若人物來自名片或使用者提供的人物照：

- 不得重新生成
- 不得重畫
- 不得變臉
- 不得美化成不同真人
- 不得卡通化
- 不得 3D 化
- 不得改變五官
- 不得改變臉型
- 不得改變髮型
- 不得改變膚色
- 不得改變身形
- 不得改變表情
- 不得改變人物角度

人物只能以「原始照片裁切貼入」方式呈現。

若無法保持人物原樣：

- 不要生成新人物
- 改為保留人物照片預留區
- 或只放聯絡資訊，不放人物

禁止為了版面完整而生成新的房仲人物。

────────────────────
三、掃碼預留區硬限制
────────────────────

掃碼圖樣一律不生成。

不得出現：
- QR Code 圖形
- LINE QR Code
- 條碼
- 類似掃碼圖樣
- 大型掃碼卡片
- 大型黑白方塊圖樣
- 大型色塊掃碼區

只能在「人物聯絡資訊區右下角」保留小型掃碼預留位。

尺寸限制：
- 寬度不得超過整張圖寬度 10%
- 高度不得超過聯絡資訊區高度 30%
- 不得大於電話資訊區
- 不得大於人物照片寬度的 35%
- 不得形成獨立大型區塊
- 不得成為視覺焦點

呈現方式：
- 細框
- 淡灰或淡米色框線
- 小字
- 留白
- 不使用厚重底色
- 不使用大面積黑底或紅底

區塊文字只能顯示：
- 掃碼預留區
或
- 加入官方通訊

用途：
僅供後續人工貼上，不是圖卡主視覺元素。

────────────────────
四、圖卡內容
────────────────────

主標：
{主標}

副標：
{副標}

價格：
{價格，若未提供則不要放}

賣點：
1. {賣點一}
2. {賣點二}
3. {賣點三}

────────────────────
五、底部固定揭露資訊區
────────────────────

圖卡底部必須清楚放入以下資訊：

聯絡人：
{聯絡人姓名，若有辨識到則放入}

電話：
{電話，若有辨識到則放入}

店名：
{完整店名，若有辨識到則放入}

經紀業名稱：
{經紀業名稱，必須放}

經紀人姓名：
{經紀人姓名，必須放}

經紀人證號：
{經紀人證號，必須放}

營業員姓名：
{營業員姓名，若有辨識到必須放}

營業員證號：
{營業員證號，若有辨識到必須放}

若已辨識到營業員姓名與營業員證號：

不得省略。
不得只讀取不顯示。
不得用「證號」模糊帶過。
不得與經紀人證號混淆。
不得互相取代。

若同時有經紀人證號與營業員證號：

兩者皆需顯示。

聯絡資訊區不得只放電話與姓名。

若空間不足，優先縮小：
- 掃碼預留區
- 裝飾元素
- icon
- 副標
- 圖片留白

不得刪除：
- 經紀業名稱
- 經紀人姓名
- 經紀人證號
- 營業員姓名（若已辨識到）
- 營業員證號（若已辨識到）

────────────────────
六、版面配置
────────────────────

建議版面結構：

上方：
- 主標
- 副標
- 價格（限有提供時放入）

中間：
- 使用真實物件照片
- 可使用多張物件照片拼接
- 不得補生成不存在畫面

下方：
- 最多 3 個核心賣點
- 人物形象照（限原圖裁切貼入）
- 聯絡資訊
- 小型掃碼預留區

最下方或底部資訊區：
- 經紀業名稱
- 經紀人姓名
- 經紀人證號
- 營業員姓名（若有辨識到）
- 營業員證號（若有辨識到）

版面原則：
- 清楚
- 留白適中
- 文字層級明確
- 手機上好閱讀
- 不讓掃碼預留區過大
- 不讓裝飾元素壓過揭露資訊

────────────────────
七、生成提醒
────────────────────

本圖卡重點是：
- 真實資訊
- 清楚排版
- 正確揭露資訊
- 真實人物呈現
- 小型掃碼預留位

不是：
- 幻想建築畫面
- 重畫人物
- 生成假掃碼圖
- 省略營業員證號
- 用風格掩蓋資訊
