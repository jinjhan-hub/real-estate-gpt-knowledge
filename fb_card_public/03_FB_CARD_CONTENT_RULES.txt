# FB 4:5 圖卡內容規則

圖卡比例：4:5。
主標：16字內。
賣點：最多3個。
價格：有提供才可放。
電話與 LINE：有提供且可辨識才可放。

必要揭露：
經紀業名稱：disclosure.brokerageName
經紀人姓名：disclosure.brokerName
經紀人證號：disclosure.brokerLicenseNo

若有營業員姓名與營業員證號，也必須放入底部資訊區。
不得用營業員資料取代經紀人資料。

禁止用語：
保證增值、穩賺不賠、必漲、唯一最低、全區最低、秒殺、投資一定賺、內定學區、未確認的走路幾分鐘。

QR Code：
一律不生成，只能小型預留區。
建議文字：QR Code 預留區 / 加入官方通訊。

版面建議：
上方：主標、副標、價格
中間：使用者提供的真實物件照片
下方：3個賣點、聯絡資訊、人物照、QR預留區
底部：經紀業、經紀人、經紀人證號、營業員資訊
