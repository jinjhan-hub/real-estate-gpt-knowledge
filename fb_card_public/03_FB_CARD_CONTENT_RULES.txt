# FB 4:5 圖卡內容規則

圖卡比例：4:5。
主標：16字內。
賣點：最多3個。
價格：有提供才可放。
電話與 LINE：有提供且可辨識才可放。

必要揭露：
經紀業名稱：disclosure.brokerageName
經紀人姓名：disclosure.brokerName
經紀人證號：disclosure.brokerLicenseNo

若有營業員姓名與營業員證號，也必須放入底部資訊區。
不得用營業員資料取代經紀人資料。

禁止用語：
保證增值、穩賺不賠、必漲、唯一最低、全區最低、秒殺、投資一定賺、內定學區、未確認的走路幾分鐘。

QR Code：
一律不生成，只能小型預留區。
建議文字：QR Code 預留區 / 加入官方通訊。

【QR Code 預留區版面規則】

正式 FB 銷售圖卡必須保留 QR Code 放置區。

位置：
下方聯絡資訊區右側或右下角。

樣式：
小型白底或淺色底方框，淡灰細框，內文字為「QR Code 預留區」。

限制：
- 不生成 QR Code
- 不生成條碼
- 不生成黑白方格
- 不生成類似掃碼圖樣
- 不得遮住公版揭露資訊
- 不得遮住營業員資訊
- 不得成為主視覺

即使使用者未提供 QR Code，也必須保留此區塊，方便後續人工貼上。

建議版面結構：

上方：
- 主標
- 副標
- 價格，限有提供時使用

中間：
- 只能使用使用者提供的真實物件照片
- 使用者提供幾張，就最多只能使用幾張
- 不得自行補出任何室內照、外觀照、街景、社區照或背景照片
- 若只有一張照片，不得做多圖拼貼
- 若照片不足，用色塊、資訊卡、icon、線條、留白與文字排版補足

下方：
- 最多 3 個核心賣點
- 聯絡資訊
- 人物形象照，限使用者提供
- 小型掃碼預留區

最下方或底部資訊區：
- 經紀業名稱
- 經紀人姓名
- 經紀人證號
- 營業員姓名，若有辨識到
- 營業員證號，若有辨識到

版面原則：
- 文字需大
- 資訊需清楚
- 不堆疊過多文字
- 不得因照片不足而生成假照片
- 不得因美觀新增未提供照片
- 不得因美觀刪除必要揭露資訊
