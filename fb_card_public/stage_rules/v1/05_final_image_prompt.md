# 05_final_image_prompt｜A-1R-3 真實素材嵌入式排版修正版

## 本階段任務

本階段負責產生最終圖片生成文字包 `final_image_prompt`。

`final_image_prompt` 的目的不是讓 AI 重新生成房屋照片或業務人物，而是讓 GPTs 對話端依照已確認資料，製作一張 FB 4:5 房仲銷售圖卡。

本階段必須明確鎖定素材來源：

1. 物件照片只能使用使用者上傳的原始真實物件照片。
2. 名片上若有人物照片，該照片即視為已提供人物素材，且是唯一合法人物來源。
3. QR Code 只能預留空白區，不得生成、重繪或仿製 QR Code。
4. 聯絡與揭露資料必須完整顯示。

---

## final_image_prompt 必須包含的最高原則

產生 `final_image_prompt` 時，必須加入以下語意：

```txt
請製作一張 FB 4:5 房仲銷售圖卡。

注意：這不是房屋照片生成任務，而是圖卡排版任務。

房屋照片區只能使用使用者上傳的原始物件照片。
請將原始物件照片嵌入版型中，可裁切、拼貼、加框、加陰影、加遮罩、調整亮度與對比，但不得重新生成、重繪、幻想或替換房屋、室內、外觀、庭院、大門、車庫、家具或裝潢。

若名片上有人物照片，該照片視為已提供人物素材。
請只使用名片上的人物照片作為業務人物唯一來源，可裁切為頭像、半身照或聯絡區人物照。
不得生成新的女性房仲、職業照、頭像、半身照或任何與名片不同的人物。
不得使用人像 icon 取代名片人物照。

QR Code 區只能預留乾淨空白框，可標示「QR Code 預留區」。
禁止生成 QR Code、仿 QR Code 或任何類 QR Code 黑白方塊圖樣。
```

## 物件照片規則

### 必須使用真實照片嵌入式排版

若使用者已上傳物件照片，`final_image_prompt` 必須明確要求：

```txt
本圖卡採真實物件照片嵌入式排版。
主要視覺照片區只能使用使用者上傳的原始物件照片。
```

### 允許處理方式

允許：

- 裁切原始照片
- 拼貼原始照片
- 放入版型
- 加框
- 加陰影
- 加漸層遮罩
- 調整亮度
- 調整對比
- 調整銳利度
- 配合圖卡排版做局部遮罩

### 禁止處理方式

禁止：

- 重新生成房屋外觀
- 重新生成室內空間
- 重新生成庭院
- 重新生成大門
- 重新生成車庫
- 重新生成家具
- 重新生成裝潢
- 依照物件資料幻想生成房屋照片
- 用 AI 生成豪宅照取代原始照片
- 將物件照改成不存在的精品豪宅風格

### luxury_premium 限制

`luxury_premium` 只能影響：

- 字體
- 色塊
- 框線
- 留白
- 排版
- 圖卡質感
- 資訊層級

`luxury_premium` 不得影響：

- 物件照片內容
- 房屋外觀
- 室內格局
- 裝潢樣貌
- 家具配置
- 建物實際狀態

## 人物照片規則

### 名片人物照視為有效人物素材

若名片上有人物照片，`final_image_prompt` 必須明確要求：

```txt
名片上的人物照片視為已提供人物素材。
請使用名片上的人物照片作為業務人物唯一來源。
```

### 允許處理方式

允許：

- 從名片裁切人物照
- 將名片人物照放入聯絡資訊區
- 製作圓形頭像
- 製作半身裁切
- 加框
- 去背
- 柔化邊緣
- 配合圖卡做自然排版

### 禁止處理方式

禁止：

- 生成新人物
- 生成女性房仲形象
- 生成職業照
- 生成頭像
- 生成半身照
- 生成與名片不同的人物
- 使用人像 icon 取代人物
- 更換人物性別
- 改變年齡感
- 改變五官
- 改變髮型
- 改變臉型
- 改變整體外貌

---

## QR Code 預留區規則

`final_image_prompt` 必須明確要求：

```txt
QR Code 區只能預留乾淨空白框。
可標示「QR Code 預留區」。
禁止生成 QR Code。
禁止生成仿 QR Code。
禁止生成任何類 QR Code 黑白方塊圖樣。
```

---

## 聯絡與揭露資料規則

`final_image_prompt` 必須要求圖卡顯示以下資料：

1. 營業員姓名
2. 聯絡電話
3. 營業員證號
4. 經紀業名稱
5. 經紀人名稱
6. 經紀人證號

若資料來源已存在，不得省略。

## A-1R-3 建議 final_image_prompt 結構

產生最終 prompt 時，建議採用以下結構：

```txt
請製作一張 FB 4:5 房仲銷售圖卡。

【任務性質】
這不是房屋照片生成任務，而是銷售圖卡排版任務。

【風格】
luxury_premium：高質感精品感、穩重、乾淨、深色或中性色基調、留白充足、文字層級清楚。
此風格只能影響字體、色塊、框線、排版與整體質感，不得改變照片內容。

【物件照片】
所有房屋照片區只能使用使用者上傳的原始物件照片。
請採真實照片嵌入式排版。
可裁切、拼貼、加框、加陰影、加遮罩，但不得重新生成、重繪、幻想或替換房屋、室內、外觀、庭院、大門、車庫、家具或裝潢。

【人物照片】
若名片上有人物照片，該照片即為唯一合法人物來源。
請只使用名片上的人物照片作為業務人物，可裁切為頭像、半身照或聯絡區人物照。
禁止生成新的業務人物、女性房仲形象、職業照、頭像或半身照。
禁止使用人像 icon 取代。

【QR Code】
僅預留乾淨空白框，可標示「QR Code 預留區」。
禁止生成 QR Code、仿 QR Code 或類 QR Code 黑白方塊圖樣。

【聯絡與揭露資料】
必須顯示營業員姓名、聯絡電話、營業員證號、經紀業名稱、經紀人名稱、經紀人證號。
```

## A-1R-3 失敗預防重點

若 prompt 中出現以下語意，必須避免：

- 生成豪宅室內照
- 生成精品住宅照片
- 生成高級客廳
- 生成房仲人物形象
- 生成女性業務頭像
- 生成 QR Code

應改成：

- 嵌入原始物件照片
- 使用上傳照片排版
- 裁切名片人物照
- 預留 QR Code 空白框
