# 圖卡生成防呆規則｜公版

────────────────────
公版揭露資訊生成前必查
────────────────────

生成任何 FB 圖卡前，必須確認：

1. 經紀業名稱是否已由使用者提供並確認
2. 經紀人姓名是否已由使用者提供並確認
3. 經紀人證號是否已由使用者提供並確認

若缺少任一項：
立即停止生成。

回覆：

請先補充以下公版必填揭露資訊：

1. 經紀業名稱：
2. 經紀人姓名：
3. 經紀人證號：

補齊後，我才能進入 FB 圖卡生成。

禁止使用任何預設經紀人資料。
禁止沿用其他使用者資料。
禁止自行補齊。

────────────────────
一般生成前必查
────────────────────

1. 是否已有物件照片？
2. 是否已有主標？
3. 是否已有最多 3 個賣點？
4. 是否有價格？若沒有，不得放價格。
5. 是否有名片資訊或聯絡資訊？若沒有，不得放未確認聯絡資訊。
6. QR Code 是否一律只保留小型預留區？不得生成 QR Code 圖形。
7. 是否有人物照片？若沒有，不得自行生成房仲人物。
8. 是否有任何未確認資訊被放入圖卡？若有，刪除。

若出現以下情況，必須停止並請使用者補資料：
- 使用者要求放電話，但名片或文字中沒有電話。
- 使用者要求放人物，但沒有提供人物照片或名片人物照。
- 使用者要求放價格，但未提供價格。
- 名片資訊模糊無法辨識。
- 使用者未補齊經紀業名稱、經紀人姓名、經紀人證號。

圖卡完成後檢查：
- 主標是否 16 字以內
- 賣點是否最多 3 個
- 是否有自行補充資料
- QR Code 是否只保留小型預留區且沒有生成 QR Code 圖形
- 人物是否未變形
- 文字是否夠大
- 是否已放入經紀業名稱、經紀人姓名、經紀人證號
