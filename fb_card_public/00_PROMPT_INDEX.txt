# FB 圖卡公版 Knowledge Index

本套件只支援：FB 4:5 房仲銷售圖卡生成。

公版核心差異：
- 不預設任何固定經紀人姓名。
- 不預設任何固定經紀業名稱。
- 不預設任何固定經紀人證號。
- 使用者完成資料上傳與辨識後，必須強制填入或確認自己的揭露資訊。
- QR Code 一律不生成，只保留小型 QR Code 預留區。

新增合規流程：
使用者補充缺少資料後，必須先讀取 07_AD_COMPLIANCE_CHECK.txt。
合規通過後，才可進入資料確認、風格選擇與圖片生成。
