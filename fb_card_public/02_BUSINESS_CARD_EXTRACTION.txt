# 名片與聯絡資訊讀取規則｜公版

目的：讀取名片或使用者提供的聯絡資訊，放入 FB 銷售圖卡下方資訊區。

本檔為公版使用。
不得預設任何固定經紀人、經紀業名稱或經紀人證號。

必讀欄位：
- 姓名
- 職稱
- 電話
- LINE ID
- 完整店名
- 經紀業名稱
- 經紀人姓名與證號
- 營業員姓名與證號
- QR Code
- 人物形象照

規則：
1. 可辨識才可使用。
2. 看不清楚就標示「未確認」，不得猜測。
3. QR Code 不得自行生成。所有 FB 圖卡一律只預留 QR Code 放置區。
4. QR Code 區塊不得出現任何 QR Code 圖形、LINE QR Code、條碼或類似掃碼圖樣。
5. QR Code 區塊只能顯示「QR Code預留區」或「加入官方LINE」。
6. 人物照片不得改變五官、臉型、膚色、髮型、身形與辨識特徵。
7. 不得將名片上的人物改成插畫、卡通、3D 假人或不同真人。
8. 若使用者指定使用名片上的人物，應盡量保留原人物位置與比例。

────────────────────
公版揭露資訊必填規則
────────────────────

每張 FB 銷售圖卡生成前，必須確認以下三項：

1. 經紀業名稱
2. 經紀人姓名
3. 經紀人證號

若名片或文字資料中可辨識，請整理為已確認資訊，並請使用者確認。

若無法辨識或缺少，必須列入需要補充資料。

缺少任一項時：
- 不得進入圖卡生成。
- 不得自行猜測。
- 不得以店名替代經紀業名稱。
- 不得以營業員姓名替代經紀人姓名。
- 不得以營業員證號替代經紀人證號。

禁止自動帶入：
- 任何未由使用者提供的經紀業名稱
- 任何未由使用者提供的經紀人姓名
- 任何未由使用者提供的經紀人證號
- 其他使用者或其他店家的固定揭露資訊

────────────────────
經紀人證號辨識規則
────────────────────

經紀人證號常見格式：
- XXX年○縣字XXXXX號
- XXX年○市字XXXXX號
- XXX○縣字XXXXX號
- XXX○市字XXXXX號

若名片或文字資料出現上述格式，優先判斷為：
經紀人證號。

若無法確認是經紀人證號或營業員證號，必須標示不確定並請使用者確認。

────────────────────
營業員證號辨識規則
────────────────────

若名片內出現：
- 登字第XXXXXX號
- (XXX)登字第XXXXXX號

視為：
營業員證號。

常見格式：
- (109)登字第XXXXXX號
- (111)登字第XXXXXX號
- 登字第XXXXXX號

不得誤判為：
經紀人證號。

營業員證號可以放入圖卡，但不得取代公版必填的經紀人證號。

若名片上同時出現姓名與登字號，請自動整理為：

營業員姓名：
依名片可辨識姓名填入

營業員證號：
依名片可辨識登字號填入

列入：
已確認資訊。

不得列入：
- 未提供
- 待補充
- 需確認
- 資訊不足

────────────────────
名片檢查輸出格式
────────────────────

- 完整店名：有 / 無 / 不清楚
- 經紀業名稱：有 / 無 / 不清楚
- 經紀人姓名與證號：有 / 無 / 不清楚
- 營業員姓名與證號：有 / 無 / 不清楚
- 電話：有 / 無 / 不清楚
- QR Code：一律預留，不生成
- 人物形象照：有 / 無

若缺少以下任一項：
1. 經紀業名稱
2. 經紀人姓名
3. 經紀人證號

必須停止生成並要求使用者補充。
