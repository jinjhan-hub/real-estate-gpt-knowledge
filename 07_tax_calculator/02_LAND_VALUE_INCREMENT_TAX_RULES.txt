# LAND VALUE INCREMENT TAX RULES｜土地增值稅試算器規格

## 頁面檔名

`land-tax-calculator.html`

## 模組定位

本模組用於保存「土地增值稅試算器」的功能規格與介面需求。

本工具僅供初步試算與教學參考，不保證實際稅額。  
實際稅額、適用稅率、土地現值、物價指數、減免條件與核定結果，仍需依稅捐機關認定為準。

---

## 主要輸入欄位

### 土地基本資料

- 土地類型：都市 / 非都市
- 土地宗地面積（平方公尺）
- 公告現值（元／每平方公尺）
- 前次移轉現值（元／每平方公尺）
- 前次移轉年月
- 移轉或設典範圍：分子 / 分母
- 消費者物價指數 dropdown

### 欄位輸入限制

- 面積需為正數。
- 公告現值需為正數。
- 前次移轉現值需為正數。
- 移轉或設典範圍分母不得為 0。
- 移轉或設典範圍不得大於全部持分，除非使用者明確說明。
- 前次移轉年月需為有效年月。
- CPI 未選擇時，不得直接試算。

---

## 版面順序

使用者偏好的欄位順序：

1. 土地類型
2. 土地宗地面積
3. 公告現值
4. 移轉或設典範圍
5. 前次移轉年月
6. 前次移轉現值
7. 消費者物價指數

---

## CPI 規則

- 需使用稅務專用物價指數概念。
- CPI 應以稅捐機關或主管機關適用資料為準。
- 若串接政府 API，需確認資料欄位是否與稅捐機關試算工具一致。
- 若 CPI dropdown 使用內建資料，需標示資料來源與更新日期。
- 若使用者指出與財政部或地方稅務局工具不同，需優先檢查：
  1. CPI 是否一致
  2. 面積是否一致
  3. 移轉或設典範圍是否一致
  4. 公告現值是否使用每平方公尺金額
  5. 前次移轉現值是否使用每平方公尺金額
  6. 土地類型是否正確
  7. 公式與級距是否一致
  8. 是否有自用住宅、重購退稅或其他減免條件未納入

---

## 計算邏輯原則

試算器需清楚呈現每個步驟，不只顯示最後稅額。

結果區建議顯示：

- 土地類型
- 宗地面積
- 移轉或設典範圍
- 本次移轉面積
- 公告現值
- 前次移轉現值
- 消費者物價指數
- 漲價總數額
- 課稅基礎
- 適用級距
- 預估土地增值稅
- 注意事項
- 免責提醒

---

## 自用住宅與減免條件

若未來加入自用住宅稅率或其他減免條件，需獨立處理，不可直接混入一般試算邏輯。

需注意：

- 不得只因使用者勾選自用住宅，就直接套用優惠稅率。
- 自用住宅條件需另列確認欄位。
- 重購退稅、農地、公共設施保留地、繼承、贈與等特殊情況，不得用一般公式直接推算。
- 若條件不完整，需顯示「特殊稅率或減免條件尚未完整確認」。

---

## 與官方試算結果不同時

若使用者提供官方試算結果作比對，不得直接假設官方錯誤。

應優先檢查：

1. 公告現值是否輸入正確
2. 前次移轉現值是否輸入正確
3. 面積是否為平方公尺
4. 移轉範圍分子 / 分母是否正確
5. CPI 是否一致
6. 前次移轉年月是否一致
7. 土地類型是否一致
8. 是否有特殊減免或自用住宅條件
9. 稅率級距或公式是否已更新

---

## 列印需求

- A4 版面
- 可另存 PDF
- 列印版需白底
- 不要深色背景
- 結果清楚可讀
- 表格不可超出頁面
- 浮水印需在最上層，但不得遮住重要數字

浮水印文字：

- 太平洋房屋站前店土地增值稅試算
- 巫建穎製作

若使用者更新品牌、店名或姓名，應以使用者最新提供內容為準。

## GitHub Pages 需求

- 頁面需可直接部署於 GitHub Pages。
- 優先使用單檔 HTML / CSS / JavaScript。
- 不依賴後端服務，除非使用者明確要求 API 串接。
- 檔名固定為 `land-tax-calculator.html`，除非使用者明確要求更改。
- 若新增入口頁，需確認連結路徑正確。

---

「本工具僅供初步試算與教學參考，實際土地增值稅額、適用稅率、物價指數、減免條件與核定結果，仍需依稅捐機關認定為準。」

---

## 禁止事項

- 不得保證稅額正確。
- 不得保證適用自用住宅優惠稅率。
- 不得保證可減免或退稅。
- 不得自行更新 CPI、稅率或公式後宣稱為最新版本。
- 不得刪除列印、PDF、浮水印與免責提醒功能。
- 不得在未確認欄位完整時輸出正式試算結果。
