# LAND VALUE INCREMENT TAX RULES｜土地增值稅試算器規格

## 頁面檔名

`land-tax-calculator.html`

## 主要輸入欄位

- 土地類型：都市 / 非都市
- 土地宗地面積（平方公尺）
- 公告現值（元／每平方公尺）
- 前次移轉現值（元／每平方公尺）
- 前次移轉年月
- 移轉或設典範圍：分子 / 分母
- 消費者物價指數 dropdown

## 版面順序

使用者偏好的順序：

1. 土地類型
2. 土地宗地面積
3. 公告現值
4. 移轉或設典範圍
5. 前次移轉年月
6. 前次移轉現值
7. 消費者物價指數

## CPI 規則

- 需使用稅務專用物價指數概念。
- 若串接政府 API，需確認資料欄位是否與稅捐機關試算工具一致。
- 若與財政部工具不同，需優先檢查 CPI、面積、移轉範圍與公式。

## 列印需求

- A4
- 浮水印：太平洋房屋站前店土地增值稅試算 / 巫建穎製作
- 浮水印在最上層
- 不要深色背景
- 結果清楚可讀
