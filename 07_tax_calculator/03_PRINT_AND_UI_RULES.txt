# PRINT AND UI RULES｜列印與介面規格

## Mobile-first

- 手機上欄位要大。
- 按鈕距離要足夠，避免誤觸。
- 帶入範例、清除、開始試算三個按鈕不可太靠近。

## 結果區

- 結果要在同頁清楚呈現。
- 列印按鈕放在試算結果最下方。
- 結果可直接列印或另存 PDF。

## A4 列印

- 使用白底。
- 不要深色背景。
- 字體大小足夠。
- 表格不可超出頁面。
- 浮水印需在最上層，但不得遮住重要數字。

## GitHub Pages

- 靜態 HTML / CSS / JS 優先。
- 入口頁可連結：
  - `index.html`
  - `land-tax-calculator.html`
- 檔名避免中文與空格。
