# 名片與聯絡資訊讀取規則

目的：讀取名片或使用者提供的聯絡資訊，放入 FB 銷售圖卡下方資訊區。

必讀欄位：
- 姓名
- 職稱
- 電話
- LINE ID
- 完整店名
- 經紀業名稱
- 經紀人姓名與證號
- 營業員姓名與證號
- QR Code
- 人物形象照

規則：
1. 可辨識才可使用。
2. 看不清楚就標示「未確認」，不得猜測。
3. QR Code 不得自行生成。未提供時只預留位置。
4. 人物照片不得改變五官、臉型、膚色、髮型、身形與辨識特徵。
5. 不得將名片上的人物改成插畫、卡通、3D 假人或不同真人。
6. 若使用者指定使用名片上的人物，應盡量保留原人物位置與比例。

名片檢查輸出格式：
- 完整店名：有 / 無 / 不清楚
- 經紀業名稱：有 / 無 / 不清楚
- 經紀人姓名與證號：有 / 無 / 不清楚
- 營業員姓名與證號：有 / 無 / 不清楚
- 電話：有 / 無 / 不清楚
- QR Code：有 / 無，若無則預留
- 人物形象照：有 / 無
