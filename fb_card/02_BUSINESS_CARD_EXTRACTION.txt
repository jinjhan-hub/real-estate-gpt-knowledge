# 名片與聯絡資訊讀取規則

目的：讀取名片或使用者提供的聯絡資訊，放入 FB 銷售圖卡下方資訊區。

必讀欄位：
- 姓名
- 職稱
- 電話
- LINE ID
- 完整店名
- 經紀業名稱
- 經紀人姓名與證號
- 營業員姓名與證號
- QR Code
- 人物形象照

規則：
1. 可辨識才可使用。
2. 看不清楚就標示「未確認」，不得猜測。
3. QR Code 不得自行生成。所有 FB 圖卡一律只預留 QR Code 放置區。
4. QR Code 區塊不得出現任何 QR Code 圖形、LINE QR Code、條碼或類似掃碼圖樣。
5. QR Code 區塊只能顯示「QR Code 預留區」或「請加入官方 LINE」。
6. 人物照片不得改變五官、臉型、膚色、髮型、身形與辨識特徵。
7. 不得將名片上的人物改成插畫、卡通、3D 假人或不同真人。
8. 若使用者指定使用名片上的人物，應盡量保留原人物位置與比例。

用途：

上述三項屬於固定揭露資訊。

不要列入：
- 待補充
- 需確認
- 資訊不足
- 未提供

若使用者提供不同資訊：
以使用者提供為優先。

輸出時直接視為：

不得自行修改證號內容。

名片檢查輸出格式
────────────────────

- 完整店名：有 / 無 / 不清楚
- 經紀業名稱：有 / 無 / 不清楚
- 經紀人姓名與證號：有 / 無 / 不清楚
- 營業員姓名與證號：有 / 無 / 不清楚
- 電話：有 / 無 / 不清楚
- QR Code：一律預留，不生成
- 人物形象照：有 / 無
