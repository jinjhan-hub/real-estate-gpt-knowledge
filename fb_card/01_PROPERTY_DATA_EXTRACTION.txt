# 物件資料讀取規則

目的：從使用者提供的圖片或文字中，整理可放入 FB 銷售圖卡的資訊。

可讀取欄位：
- 案名
- 開價 / 總價
- 地址或區域
- 土地坪數
- 建物坪數
- 格局
- 樓層
- 屋齡
- 車位
- 類型，例如透天、別墅、大樓、華廈、公寓、店面、土地
- 特色，例如大地坪、可停多車、雙併、邊間、電梯、孝親房
- 使用者明確提供的生活機能或交通資訊

禁止：
- 不得自行補價格、坪數、地址、格局、屋齡。
- 不得從照片推測學區、交通距離、生活機能。
- 不得自行推測屋況，例如全新整理、可立即入住，除非使用者提供。

資料確認輸出格式：
1. 已確認資訊
2. 不確定資訊
3. 不能自行補充的資訊
4. 需要使用者補充的資料
5. 是否可以進入圖卡生成
