# FB 圖卡圖片生成提示詞模板

使用時請根據已確認資訊替換括號內容。

生成一張 Facebook 4:5 房仲銷售圖卡。

風格：{風格名稱，例如高級豪宅風 / 溫暖家庭風 / 乾淨白底商務風}

限制：
1. 只能使用使用者提供的物件照片、名片與人物形象照。
2. 不要自行生成建物外觀、室內照片、街景或地圖。
3. 不要自行補價格、坪數、地址、電話、學區、交通距離。
4. QR Code 未提供時，只做 QR Code 預留區。
QR Code 規則：

QR Code 一律不生成。

僅於右下角保留小型 QR Code 放置區。

尺寸：

約人物名片區高度的 25%～30%。

不得大於人物資訊區。

不得獨立形成大型區塊。

僅顯示：

「QR Code 預留區」

或：

「加入官方LINE」

以細框、淡色、小面積呈現。

QR Code 區域用途為後續人工貼入。

不是版面主視覺。
5. 人物若來自名片或形象照，需保留五官、臉型、身形與辨識特徵，不得變臉。
6. 文字要大，手機上清楚可讀。
7. 主標 16 字以內，賣點最多 3 個。
8. 整體像房仲銷售圖卡，乾淨、清楚，不要過度浮誇。

圖卡內容：
主標：{主標}
副標：{副標}
價格：{價格，若未提供則不要放}
賣點：
1. {賣點一}
2. {賣點二}
3. {賣點三}

下方聯絡資訊：
{姓名 / 電話 / 店名 / 經紀業名稱 / 證號，僅放已確認資訊}

版面：
上方放主標與副標；中間使用物件照片；下方放三個賣點、聯絡資訊、人物形象照與 QR Code 預留區。整體排版清楚、留白適中、文字層級明確。
