# FB 圖卡圖片生成提示詞模板

使用時請根據已確認資訊替換括號內容。

生成一張 Facebook 4:5 房仲銷售圖卡。

風格：{風格名稱，例如高級豪宅風 / 溫暖家庭風 / 乾淨白底商務風}

限制：
1. 只能使用使用者提供的物件照片、名片與人物形象照。
2. 不要自行生成建物外觀、室內照片、街景或地圖。
3. 不要自行補價格、坪數、地址、電話、學區、交通距離。
4. 人物若來自名片或形象照，需保留五官、臉型、身形與辨識特徵，不得變臉。
5. 文字要大，手機上清楚可讀。
6. 主標 16 字以內，賣點最多 3 個。
7. 整體像房仲銷售圖卡，乾淨、清楚，不要過度浮誇。

────────────────────
QR Code 版面硬限制
────────────────────

QR Code 一律不生成。

不得出現：
- QR Code 圖形
- LINE QR Code
- 條碼
- 類似掃碼圖樣
- 大型 QR Code 卡片
- 大型 QR Code 色塊

只能在「人物聯絡資訊區右下角」保留小型 QR Code 放置位。

尺寸限制：
- 寬度不得超過整張圖寬度 10%
- 高度不得超過人物聯絡資訊區高度 30%
- 不得大於電話號碼區
- 不得大於人物照片寬度的 35%
- 不得形成獨立大型區塊
- 不得成為視覺焦點

呈現方式：
- 細框
- 淡灰或淡米色框線
- 小字
- 留白
- 不使用厚重底色
- 不使用大面積黑底或紅底

區塊文字只能顯示：
QR Code預留區

或：
加入官方LINE

QR Code 區域用途：
僅供後續人工貼上，不是圖卡主視覺元素。

圖卡內容：
主標：{主標}
副標：{副標}
價格：{價格，若未提供則不要放}
賣點：
1. {賣點一}
2. {賣點二}
3. {賣點三}

下方聯絡資訊：
{姓名 / 電話 / 店名 / 經紀業名稱 / 證號，僅放已確認資訊}

版面：
上方放主標與副標。
中間使用物件照片。
下方放人物形象照、姓名、電話、店名、經紀業名稱、經紀人姓名與證號。
QR Code 預留區只能作為人物聯絡資訊區右下角的小型輔助區塊，不得獨立成大型卡片。
整體排版清楚、留白適中、文字層級明確。
