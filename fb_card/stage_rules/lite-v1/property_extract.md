# property_extract｜物件資料整理與確認

## 階段任務

將使用者本輪提供的素材整理成房仲銷售圖卡需要的物件資料，並請使用者確認。未經確認的欄位不得進入後續產圖摘要。

## 可整理欄位

請整理下列欄位，沒有資料時填「待補」或「不顯示」：

- 案名或主標
- 縣市、行政區、商圈或路段
- 物件類型：電梯大樓、公寓、透天、別墅、華廈、店面、土地、廠辦等
- 總價或開價
- 建物坪數
- 土地坪數
- 主建物、附屬建物、共有部分
- 格局
- 樓層與總樓層
- 屋齡
- 車位
- 朝向
- 管理費
- 學區、交通、商圈與生活機能
- 三到五個銷售亮點
- 適合客群：首購、換屋、置產、收租、企業主、退休族等

## 文案整理規則

- 使用台灣房仲常用語，保持清楚、好讀、可信。
- 主標應短、明確、有記憶點，但不得誇大。
- 賣點可以重寫成圖卡文案，但不得新增不存在的設施、捷運站、學區或商圈。
- 價格、坪數、屋齡、格局、車位等數字必須忠於來源。
- 若來源互相衝突，列出衝突並請使用者確認。
- 不得使用「保證增值」、「穩賺不賠」、「唯一最低」、「全台最強」等絕對或保證性文字。

## 請使用者確認

完成整理後，必須請使用者確認，格式如下：

```txt
我整理到的物件資料如下，請確認是否正確：

案名：
區域：
物件類型：
總價：
坪數：
格局：
樓層：
屋齡：
車位：
主要賣點：
1.
2.
3.

待補或待確認：
- 

請回覆「確認」或直接修正需要調整的欄位。
```

## 完成條件

使用者明確確認或修正完欄位後，才可進入合規檢查。
