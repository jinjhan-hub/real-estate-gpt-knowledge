# material_check｜本輪素材狀態檢查

## 階段任務

檢查使用者本輪提供的素材是否足以製作房仲銷售圖卡，並判斷下一步應補資料、整理物件資訊或進入合規檢查。

## 必查素材

請依序檢查：

1. 物件基本資料：案名、區域、地址或地段、總價、坪數、格局、樓層、屋齡、車位、用途或類型。
2. 物件賣點：採光、格局、交通、學區、商圈、生活機能、稀有性、投資亮點。
3. 物件照片：外觀、室內、格局、社區、公設或環境照片。
4. 房仲名片或聯絡資料：營業員姓名、手機、公司或店名、營業員證號。
5. 經紀業揭露資料：經紀業名稱、經紀人姓名、經紀人證號、公司地址或電話。
6. QR Code：是否有提供原始 QR Code 圖檔；若未提供，只能規劃預留區。
7. 人物照：是否有提供真人照片，或名片上是否含可用的人物照片。

## 判斷規則

- 使用者有提供照片時，標記為「已提供」，但不得把照片內容改寫成不存在的空間。
- 名片上若有人物照片，視為本輪可用人物素材，但仍不得生成新人物。
- QR Code 若有提供，僅能視為後續人工或排版使用的原始素材；產圖提示詞不得要求重繪或仿製。
- 不清楚、模糊、被遮住或無法辨識的文字，一律標為「待確認」。
- 使用者未提供的證號、電話、地址、公司名稱，不得推測。

## 回覆格式

請用以下格式回覆使用者：

```txt
目前素材檢查：
- 物件資料：完整 / 部分缺少 / 未提供
- 物件照片：已提供 / 未提供
- 名片或聯絡資料：已提供 / 部分缺少 / 未提供
- 經紀業揭露：完整 / 部分缺少 / 未提供
- QR Code：已提供 / 未提供，將僅預留區
- 人物照：已提供 / 名片含人物照 / 未提供

需要補充或確認：
- [列出缺少或待確認項目]
```

## 阻擋條件

若缺少下列任一類資訊，應先請使用者補充或確認，不得直接進入產圖：

- 主要物件資料不足以形成圖卡主文案。
- 聯絡資料缺少營業員姓名或電話。
- 經紀業揭露資料完全缺少。
- 使用者要求放 QR Code，但未提供原始 QR Code，且不同意只預留區。
