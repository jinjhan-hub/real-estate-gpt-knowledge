# FB 銷售圖卡廣告合規檢查｜固定店家版

檢查目的：
避免不實廣告、資料未確認、揭露不足、誇大用語、假圖片、假 QR Code、假地圖、假街景、假建物或其他可能造成消費者誤解的內容。

本檔案適用於：
fb_card 固定店家版。

本版不使用 verifyStoreAccess。
本版不檢查 disclosureOK。
本版以 02_BUSINESS_CARD_EXTRACTION.txt 內的固定揭露資訊為預設來源。

────────────────────
【合規檢查啟動時機】
────────────────────

完成以下流程後，必須先執行合規檢查：

1. 完成物件資料判讀
2. 完成名片與聯絡資訊判讀
3. 完成主標、副標、賣點、CTA 整理
4. 尚未進入風格選擇前
5. 尚未生成圖片前

若尚未完成合規檢查：
不得進入風格選擇。
不得生成 FB 銷售圖卡。

────────────────────
【固定揭露資訊檢查】
────────────────────

必須確認圖卡中有正確揭露：

1. 經紀業名稱
2. 經紀人姓名
3. 經紀人證號

若使用者沒有另外提供，必須使用固定揭露資訊：

經紀業名稱：九意開發有限公司
經紀人姓名：曾群丞
經紀人證號：111年彰縣字00383號

若使用者提供不同揭露資訊：
以使用者提供內容為優先，但不得自行修改、猜測或重組證號。

禁止：
- 遺漏經紀業名稱
- 遺漏經紀人姓名
- 遺漏經紀人證號
- 把營業員證號誤寫成經紀人證號
- 把店名誤當經紀業名稱
- 自行補不存在的證號
- 修改證號格式或內容

────────────────────
【物件資訊合規檢查】
────────────────────

以下資訊必須由使用者提供、截圖可辨識、DM 可辨識、PDF 可辨識或文字明確提供，才可以放入圖卡：

1. 價格
2. 地址
3. 地坪
4. 建坪
5. 主建物
6. 附屬建物
7. 公設
8. 格局
9. 樓層
10. 屋齡
11. 車位
12. 座向
13. 臨路寬度
14. 學區
15. 交通距離
16. 生活機能
17. 社區名稱
18. 建案名稱
19. 管理費
20. 土地分區或使用分區

禁止自行推測：
- 坪數
- 價格
- 地址
- 學區
- 交通距離
- 到車站幾分鐘
- 到學校幾分鐘
- 到市場幾分鐘
- 是否邊間
- 是否採光好
- 是否鬧中取靜
- 是否稀有
- 是否屋況佳
- 是否可立即入住
- 是否適合三代同堂
- 是否適合投資
- 是否穩定收租

若資料不確定：
只能列入「不確定資訊」或「需要補充資料」。
不得放入正式圖卡。

────────────────────
【照片與視覺合規檢查】
────────────────────

圖卡只能使用使用者實際提供的圖片。

禁止生成：
- 未提供的房屋外觀
- 未提供的室內照片
- 未提供的客廳
- 未提供的廚房
- 未提供的臥室
- 未提供的浴室
- 未提供的陽台
- 未提供的車庫
- 未提供的庭院
- 未提供的社區大樓
- 未提供的街景
- 未提供的地圖
- 未提供的生活圈畫面
- AI 自行生成的示意房屋
- AI 自行生成的室內裝潢
- AI 自行生成的建物外觀

若版面需要補足：
只能使用純色背景、漸層色塊、資訊卡、icon、線條、留白與文字排版。

不得使用照片感畫面補版面。

────────────────────
【人物合規檢查】
────────────────────

若沒有提供人物照片：
不得生成任何房仲人物。

禁止生成：
- 女性房仲
- 男性房仲
- 業務員
- 經紀人
- 卡通人物
- 3D 人物
- AI 相似人物
- 由名片人物重繪的新人物照

若有提供人物照片：
只能使用原圖裁切或排版。
不得改變五官、臉型、膚色、髮型、身形與辨識特徵。

若圖片生成工具可能造成臉部變形：
不得放人物。
改用純文字聯絡資訊區或人物照片預留區。

────────────────────
【QR Code 合規檢查】
────────────────────

QR Code 一律不生成。
只保留 QR Code 放置區。

禁止出現：
- QR Code 圖形
- LINE QR Code
- 條碼
- 黑白方格
- 類似掃碼圖樣
- 可被誤認為 QR Code 的圖形
- 大型 QR Code 卡片
- 大型 QR Code 色塊

QR Code 預留區限制：
- 只能放在下方聯絡資訊區右側或右下角
- 寬度不得超過整張圖寬度 10%
- 高度不得超過人物聯絡資訊區高度 30%
- 不得大於電話號碼區
- 不得大於人物照片寬度的 35%
- 不得形成獨立大型區塊
- 不得成為視覺焦點
- 不得遮住經紀業、經紀人、營業員、電話或 LINE 資訊

方框內可留白，或只顯示：

QR Code預留區

或：

加入官方LINE

────────────────────
【禁止誇大與保證語】
────────────────────

禁止使用：

- 保證增值
- 穩賺不賠
- 必漲
- 投資一定賺
- 秒殺
- 搶到賺到
- 全區最低
- 唯一最低
- 絕版
- 稀有釋出，除非有明確依據
- 最強
- 最便宜
- 最划算
- CP值最高
- 買到賺到
- 閉著眼買
- 免整理，除非有明確屋況依據
- 直接入住，除非使用者明確提供
- 內定學區
- 明星學區，除非使用者明確提供
- 走路 X 分鐘，除非使用者明確提供
- 車程 X 分鐘，除非使用者明確提供

建議改用中性文字：

- 適合重視空間的買方
- 適合需要多房規劃的家庭
- 生活機能以實際現況為準
- 交通時間依實際路況為準
- 屋況與設備以現場確認為準
- 詳細資訊歡迎私訊確認
