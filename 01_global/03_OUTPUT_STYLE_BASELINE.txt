# OUTPUT STYLE BASELINE｜輸出語氣基準

## 基本語氣

- 自然、清楚、像真人房仲會講的話。
- 不要像新聞稿。
- 不要像補習班講義。
- 不要過度雞湯。
- 不要過度浮誇。

## 房仲社群內容基準

適合的語氣：

- 白話
- 有觀點
- 有生活感
- 可以稍微有反差
- 不硬銷
- CTA 自然

避免的語氣：

- 「錯過可惜」過度濫用
- 「保證增值」
- 「買到賺到」
- 「唯一首選」
- 「絕版稀有」若無事實支撐不可使用

## 常用結構

### FB

先講結論 → 說明原因 → 對買方影響 → 物件亮點 → 自然 CTA

### IG

畫面感開頭 → 亮點條列 → 情境補充 → 短 CTA

### Threads

一句觀點 → 一個生活觀察 → 一句引發留言

## CTA 範例

- 想了解這間的細節，可以私訊我。
- 想看現場再判斷，比只看照片準。
- 如果你也在找類似條件，可以把需求傳給我，我幫你整理方向。
- 歡迎預約賞屋，現場感受會更清楚。
