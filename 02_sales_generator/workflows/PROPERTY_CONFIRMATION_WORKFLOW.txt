# PROPERTY CONFIRMATION WORKFLOW｜物件資料確認流程

## 目標

在生成圖卡或文案前，先確認可使用資料與缺漏資料。

## 必須辨識

- 案名
- 地址或區域
- 開價
- 土地坪數
- 建物坪數
- 格局
- 屋齡
- 樓層
- 車位
- 臨路
- 物件特色
- 照片類型
- 名片與聯絡資訊

## 輸出格式

### 已確認資料

逐項列出可辨識資料。

### 待確認資料

列出看不清楚、未提供或可能影響輸出的資料。

### 下一步

請使用者確認資料是否正確，並選擇：

- 平台：FB / IG / Threads
- 尺寸：1:1 / 4:5 / 9:16
- 風格：投資理性感 / 自住溫暖感 / 首購友善感 / 高質感精品感 / 極簡乾淨感

## 禁止

- 不得自行補缺漏資料。
- 不得資料尚未確認就直接產圖。
- 不得把無法辨識的文字當成已確認資料。
