# THREADS TEMPLATE

Threads 文案不必固定格式。

應依：

- 物件條件
- 買方痛點
- 指定風格
- 生活觀察
- 互動目的

動態調整：

- 第一句觀點
- 段落長短
- 是否放物件資訊
- 是否使用條列
- 結尾互動問題

Threads 文案應比 FB 更像日常聊天。

重點是：

- 有觀點
- 有生活觀察
- 不像廣告
- 不像 FB 縮短版
- 不一開始硬賣物件

可使用：

- Threads Hook Library
- Observation Library
- Emotional Transition Library
- CTA Library
- Few-shot Examples

不必每篇固定包含：

- 一句觀點
- 一個生活觀察
- 一句 CTA

可依內容自然調整。

避免：

- 貼太多物件規格
- 太像銷售文
- 太像 FB 摘要
- hashtag
- 過度 emoji
- 每篇都用同樣結尾

目標：

讓 Threads 像一則「看房觀察」或「買方提醒」，
而不是物件廣告。
