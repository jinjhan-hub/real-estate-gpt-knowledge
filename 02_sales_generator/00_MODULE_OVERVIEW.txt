# SALES GENERATOR MODULE｜銷售圖卡與社群文案模組

本模組處理：

- 物件資料辨識
- 名片與聯絡資訊辨識
- 銷售圖卡產圖設定
- FB / IG / Threads 文案
- 平台規則、尺寸規則、風格規則與生成流程

## v0.2.5 調用原則

銷售生成不再建議一次載入整個模組。

應依下列四條線分別調用：

1. platform：FB / IG / Threads
2. ratio：1:1 / 4:5 / 9:16
3. style：投資理性感 / 自住溫暖感 / 首購友善感 / 高質感精品感 / 極簡乾淨感
4. workflow：資料確認 / 產圖設定 / 圖片生成 / 文案生成 / 最終檢查

## 核心限制

- 所有物件資訊只能使用使用者提供、可辨識、可確認的資料。
- 不得自行補價格、坪數、地址、格局、屋齡、電話、LINE、QR Code、公司名稱。
- 人物照片不得變形成另一個人。
- 名片、QR Code、Logo、聯絡資訊必須以使用者提供素材為準。
- 沒有確認資料前，不得直接生成正式銷售圖卡。
- 圖片生成階段只處理短版產圖摘要，不同時輸出完整文案。

## 建議流程

1. 資料確認
2. 產圖設定
3. 圖片生成
4. 文案生成
5. 最終檢查
